We are excited to announce the first release of CC-Bench, a modular benchmark framework for high-performance communication and compression evaluation in HPC environments.

This initial release includes:

- MPI, NCCL, and RCCL communication backend support
- LD_PRELOAD-based instrumentation wrappers
- CCL compression plugin integration
- Background daemon monitoring
- JSONC-based configuration with auto-generation scripts

Check the [Leaderboard](/leaderboard/) for initial benchmark results.
